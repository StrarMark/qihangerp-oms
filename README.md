# 启航电商OMS订单处理系统

#### 介绍
启航电商OMS订单处理系统是一套为中小电商企业构建的一套简单实用的第三方平台订单处理系统，本项目采用Java SpringBoot+Vue2前后端分离开发。

本系统支持多平台订单、售后、发货等接口对接，包括：淘宝、拼多多、抖店、快手小店、小红书等。

#### 软件架构

1、技术栈
vue2 + elementUI
SpringBoot2.x
Java 17
2、存储栈
MySQL8数据库
minio文件存储
3、中间件
Redis（缓存：在线用户、字典、系统配置）
Nacos配置中心
quartz定时任务
